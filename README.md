# Ponto de Apoio

Informação, acolhimento e orientação contra a violência doméstica.

Site informativo criado com Vite + React, CSS puro e publicação estática no GitHub Pages.

## Stack

- Vite + React
- CSS puro
- Sem backend
- Sem banco de dados
- GitHub Pages via GitHub Actions

## Rodar localmente

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy no GitHub Pages

O workflow `.github/workflows/deploy.yml` publica a pasta `dist` automaticamente quando houver push na branch `main`.

No repositório do GitHub, ative:

- Settings > Pages
- Source: GitHub Actions

## Bases de conteúdo

O conteúdo inicial foi montado a partir dos PDFs fornecidos e de fontes oficiais/organizacionais como OVM/SC, Instituto Maria da Penha, CNJ, Ministério das Mulheres, Pacto Nacional Brasil Contra o Feminicídio, Justiceiras e Instituto Nós Por Elas.

O site também inclui uma área de pesquisa com links para bases acadêmicas e fontes de estudo como SciELO, Biblioteca Virtual em Saúde, Portal de Periódicos CAPES, BDTD, OVM/SC e DataSenado/OMV.
